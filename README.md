# Hyprland Config

[![CI](https://github.com/RomeoCavazza/hyprland-config/actions/workflows/ci.yml/badge.svg)](https://github.com/RomeoCavazza/hyprland-config/actions/workflows/ci.yml)

Hyprland desktop configuration consumed by the NixOS workstation flake.

The repo carries compositor, Waybar, Rofi, foot, eDEX-UI, swappy, theme, and
helper-script configuration. It is intended to be imported by
[`nixos-config`](https://github.com/RomeoCavazza/nixos-config) rather than
applied standalone.
